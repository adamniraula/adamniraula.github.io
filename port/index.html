This course introduces web development fundamentals including HTML, CSS, and JavaScript. We learn to build interactive, responsive websites by applying modern semantic elements and styling techniques.

Assignments

Assignment 01 – Basic HTML

For the first assignment, I structured a simple web page practicing headings, paragraphs, and links.

Assignment 02 – Basic CSS

I applied CSS styling to layout and style content, experimenting with colors, fonts, and external style sheets.

Assignment 03 – Page Layout

Created a responsive single‐page layout using CSS Flexbox.

Assignment 04 – Recreate CSS Page

I recreated the University of South Carolina Swimming & Diving webpage.

Assignment 05 – JavaScript Buttons, Functions, and More

Created a responsive 3-column web page using HTML, CSS, and JavaScript. Followed class conventions including arrow functions and constants.

Assignment 06 – JavaScript: Conditionals

Created a single‐page layout with centered content and neatly aligned inputs, and used JavaScript conditionals to toggle between the “Traveling” and “Picture Chooser” sections when each exercise button is clicked.

Assignment 07 – For Loops

Used a JS for-loop to draw six CSS town houses, then animated a stick-figure stretch sequence with setInterval.

Assignment 08 – JavaScript: Arrays

Built an image gallery with hover overlays and styled popups using JavaScript arrays. Included image credits.

Assignment 09 – JavaScript: Classes

This assignment demonstrates the use of JavaScript classes to build a dynamic hot dog gallery. Hover reveals names; clicking opens a modal with details.
